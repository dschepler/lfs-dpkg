6.10. Man-pages-4.00

The Man-pages package contains over 2,200 man pages.

Approximate build time: less than 0.1 SBU
Required disk space: 35 MB

6.10.1. Installation of Man-pages

Install Man-pages to a temporary directory: 

make install DESTDIR=$(pwd)/debian/tmp

Remove a few man pages that will be provided by their own packages later: 

rm -fv debian/tmp/usr/share/man/man5/attr.5 \
       debian/tmp/usr/share/man/man3/getspnam.3 \
       debian/tmp/usr/share/man/man5/passwd.5

Create the package archive: 

buildpkg

Install the package: 

dpkg -i ../man-pages_4.00_*.deb
