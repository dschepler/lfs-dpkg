1.3. Changelog

This is version 20150524-dpkg of the Linux From Scratch book, dated May 24, 2015. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via http://www.linuxfromscratch.org/mirrors.html.
